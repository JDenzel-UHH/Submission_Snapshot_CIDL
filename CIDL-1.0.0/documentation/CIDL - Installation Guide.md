# Installation
---

CIDL is published on PyPI under the package name `cidl-uhh`.

Install it with 

bash:
```bash
pip install cidl-uhh
```

After installation, import it in Python as 

python:
```python
import cidl
```

#### Recommended check (optional)

To verify that the installation worked, run 

bash:
```bash
python -c "import cidl; print(cidl.__version__)"
```

If no error appears, CIDL was installed successfully.

#### Important note

The PyPI package name is `cidl-uhh`.

The Python import name is `cidl`.
