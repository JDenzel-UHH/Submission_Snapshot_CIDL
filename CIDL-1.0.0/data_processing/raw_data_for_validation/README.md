## raw_data_for_validation

Due to storage limitations, only datasets `0001` and `3400` are included in their original form.

The remaining `patient_year` CSV files were too large to be uploaded as single files and were therefore split into two parts. To restore the original file, the corresponding `..._PART1.csv` and `..._PART2.csv` files must be combined again, while omitting the header row contained in `..._PART2.csv`.
